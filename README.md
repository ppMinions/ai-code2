# 番茄钟

一个简洁的番茄工作法计时器，提供 Web 和桌面两种版本。

## 功能

- **三种模式**：专注（25 分钟）、短休（5 分钟）、长休（15 分钟）
- **环状进度条**：可视化剩余时间
- **自动切换**：专注结束自动进入休息，休息结束自动进入下一个专注
- **智能长休**：每完成 4 个番茄，自动切换为长休息
- **完成统计**：记录累计完成的番茄数
- **深色主题**：护眼暗色界面

## 使用方式

### Web 版

用浏览器打开 `pomodoro.html` 即可使用，数据存储在浏览器 localStorage。

### 桌面版

```bash
python pomodoro.py
```

需要 Python 3 和 tkinter（Python 标准库自带），数据存储在 `.pomodoro_data.json`。

## 文件说明

| 文件 | 说明 |
|------|------|
| `pomodoro.html` | Web 版番茄钟，纯 HTML/CSS/JS |
| `pomodoro.py` | 桌面版番茄钟，基于 tkinter |
